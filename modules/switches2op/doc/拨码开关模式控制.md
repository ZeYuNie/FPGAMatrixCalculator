# switches2op

简单的拨码开关控制模块

控制逻辑：

<!--         4'b00001: op = 3'd5; // Settings
        4'b00010: op = 3'd4; // Calculate
        4'b00100: op = 3'd3; // Show
        4'b01000: op = 3'd2; // Generate
        4'b10000: op = 3'd1; // Matrix Input
        default: op = 3'd0; // Invalid -->

| 开关状态 | 输出状态 |
| -------- | -------- |
| 00001 | setting 模式 |
| 00010 | calculate 模式 |
| 00100 | show 模式 |
| 01000 | generate 模式 |
| 10000 | matrix input 模式 |
| 其他状态 | 无效状态 |
