# seg7_display - 七段数码管显示

## 功能
驱动4位共阳极七段数码管显示BCD数据。

## 端口
| 端口 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| clk | input | 1 | 系统时钟 |
| rst_n | input | 1 | 异步复位，低电平有效 |
| valid | input | 1 | 显示有效信号 |
| bcd_data_0 | input | [3:0] | 数字0（最右侧） |
| bcd_data_1 | input | [3:0] | 数字1 |
| bcd_data_2 | input | [3:0] | 数字2 |
| bcd_data_3 | input | [3:0] | 数字3（最左侧） |
| seg | output | [7:0] | 段选信号 |
| an | output | [3:0] | 位选信号 |

## 工作原理
- 动态扫描显示：每5ms切换一位（200Hz刷新率）
- valid=0时关闭所有显示
- 支持0-9数字和字母E显示

## 段码编码
共阳极：低电平点亮
- 数字0-9: 标准7段编码
- 数字10: 显示'E'
- 其他: 全灭
