# bin_to_bcd - 二进制转BCD码

## 功能
将16位二进制数转换为4位BCD码（支持0-9999）。

## 端口
| 端口 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| bin_in | input | [15:0] | 二进制输入 |
| bcd_out | output | [3:0][0:3] | BCD输出数组 |

## BCD输出排列
- `bcd_out[0]`: 千位
- `bcd_out[1]`: 百位
- `bcd_out[2]`: 十位
- `bcd_out[3]`: 个位

## 工作原理
采用除法和取模运算：
- 个位 = bin_in % 10
- 十位 = (bin_in / 10) % 10
- 百位 = (bin_in / 100) % 10
- 千位 = (bin_in / 1000) % 10

## 特性
- 纯组合逻辑，无延迟
- 自动截断超过9999的高位
